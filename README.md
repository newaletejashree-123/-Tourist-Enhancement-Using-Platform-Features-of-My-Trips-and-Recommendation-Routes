Project Overview
This project is developed for the Teaching Assistant (TA) assignment. The objective is to study a research paper and recreate the web interface screens shown in the paper using HTML and CSS.
Research Paper

Title: Tourist Enhancement Using Platform Features of My Trips and Recommendation Routes

Domain: Digital Tourism / Travel Recommendation System
Features Implemented

Home Page
Recommendation Routes Page
Route Details Page
My Trips Page
Create Trip Page
Trip Details Page
Technologies Used
HTML
CSS

Project Structure

index.html – Home page
recommendation.html – Shows recommended travel routes
route-details.html – Displays route information
mytrips.html – Shows saved trips
create-trip.html – Page to create a trip
trip-details.html – Displays trip itinerary
css/style.css – Styling for the website

Purpose
The purpose of this project is to understand research-based system design and implement the interface screens using basic web development technologies.
